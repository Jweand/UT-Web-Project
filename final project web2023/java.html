what is java?

Java is a widely-used programming language for coding web applications.
It has been a popular choice among developers for over two decades, with millions of Java applications in use today.

Java is a multi-platform, object-oriented, and network-centric language that can be used as a platform in itself.
It is a fast, secure, reliable programming language for coding everything from mobile apps and enterprise software to big data applications and server-side technologies.

Java Syntax

Java's structure derives much from the structure of the C programming language:
assignment structure, expressions, control, and blocks.

Java was specifically designed to be an object-oriented language, where all code is contained within objects,
except for basic variables (numbers, logical values, and characters), which are not placed as objects for performance reasons.

To avoid many problems, Java does not support several properties, such as multiple inheritance and operator overloading.

Java uses the same two types of comments as C++:
end-of-line comments, which begin with "//", and multi-line comments, which begin with "*/" and end with "/*".

Java programming, of course, depends on C++ comments, as was previously known,
because it is a programming language resulting from the development of the C language
